Otras Propiedades de Display

1. Mostrar en tabla

  • display: table : permite que un elemento se comporte como una tabla (similar al elemento < table>)

  • display: table-column : permite que un elemento se comporte como la columna de la tabla (similar al elemento < col> )

  • display: table-row : permite que un elemento se comporte como la fila de la tabla (similar al elemento < tr> )

  • display: table-column-group : permite que un elemento se comporte como el grupo de columnas de la Tabla (similar al elemento < colgroup> )

  • display: table-row-group : permite que un elemento se comporte como el grupo de filas de la Tabla (similar al elemento < tbody> )

  • display: table-header-group : permite que un elemento se comporte como el encabezado de la tabla (similar al elemento < thead> )

  • display: table-footer-group : permite que un elemento se comporte como el pie de página de la tabla (similar al elemento < tfoot> )

  • display: table-cell : permite que un elemento se comporte como la celda de la Tabla (similar al elemento < td> )

  • display: table-caption : permite que un elemento se comporte como el título de la tabla (similar al elemento < caption> )
