Contenido

La propiedad "display" es la más importante para controlar estructuras. Cada elemento tiene un valor de display por defecto dependiendo de qué tipo de elemento sea.
El valor por defecto para la mayoría de los elementos es usualmente Block (de bloque) o Inline (en línea). Un elemento que es Block es comunmente llamado elemento Block-Level. Un elemento Inline siempre es llamado elemento inline.
